# 💻 What is this Tool?

<div class="card">

The **German A1.1 Interactive Learning Tool** is a digital course for **beginner German learners (A1.1 level)**.

It was created using **LiaScript**, an open-source system for interactive learning materials.

The course includes:

- 📘 Interactive lessons
- 🎧 Listening activities
- 🧠 Vocabulary practice
- 🤖 Chatbot support
- 📝 Self-assessment tests

Students can access the course directly in a **web browser**.

</div>

---

# 🧩 Structure of the Learning Tool

<div class="card">

The course is organized into **learning slides or sections**.

Each slide may contain:

- Explanations of German topics
- Listening dialogues
- Vocabulary lists
- Interactive exercises
- Practice tests

Students move through the course step by step.

</div>
