# ✍️ SCHREIBEN – Teil 3

## Im Café bestellen

<details style="background:#fff4f8;border:1px solid #efcad8;border-radius:14px;padding:14px;margin-bottom:18px;">
<summary><b>💡 Hint</b></summary>

Write a short café dialogue.

Include:

1. Greeting  
2. Ordering food/drinks  
3. Saying what you like  
4. Paying  
5. Goodbye

Useful phrases:

- Wir möchten gern bestellen.
- Ich nehme ...
- Ich mag ...
- Zusammen oder getrennt?
- Vielen Dank.

Use short A1 sentences.

</details>

 ## Schreiben Sie einen kurzen Dialog im Café.

Schreiben Sie:

- Was möchten Sie essen?
- Was möchten Sie trinken?
- Bezahlen Sie zusammen oder getrennt?
